Description Info

I wrote this book to bring the basics of programming in C# to you without cluttering the issue with too much information too soon. It focuses on console applications to bypass the tendency to jump into programming feet first, and ending up with lots of bells and whistles that doesn't really accomplish anything. It is, however, detailed eugh to start any reader on the exciting road to programming in C#. The examples are detailed eugh to give a good introduction to Object Oriented Programming (OOP), without clouding the learning process with some of the more difficult principles of OOP. The book is short eugh to be read in one sitting, but t so short that it doesn't accomplish what it sets out to do. All the resources used to do the programming shown in the book can be downloaded from the World Wide Web without any costs. I have included all the necessary links to the sources used. I really hope you enjoy reading this book just as much as I enjoyed writing it. In a few minutes you can be an your way, and who kws what the future holds for you and C#.
